O que são as Imagens em HTML?

Em HTML, a tag <img> é usada para incorporar imagens em uma página web. Ela é um elemento vazio, ou seja, não possui uma tag de fechamento, e requer pelo menos o atributo src para funcionar corretamente.

Código base: <img src="caminho/para/imagem.jpg" alt="Descrição da imagem" width="300" height="200">
