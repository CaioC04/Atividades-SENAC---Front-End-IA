Lista não ordenada

Em HTML, uma lista não ordenada é representada pela tag <ul> (do inglês Unordered List). Ela é utilizada quando a ordem dos itens não é importante, como em listas de compras, tarefas ou categorias. Cada item da lista é definido pela tag <li> (do inglês List Item).

Exemplo de uso prático:

    <li>Maçãs</li>

    <li>Bananas</li>

    <li>Uvas</li>

    </ul>
